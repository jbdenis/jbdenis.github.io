    • Scripts

    All scripts give their documentation when launched without argument.
      gefi (arbo, jeter, nettoyer, norfo, padb, retrouve, sauver).
      gefi is a general script to see and rename sets of files. Its options are too numerous and complicated, so simplified calls of it are proposed in the scripts in between the parentheses.
      mado.pl
      To generate the documentation file of a module file, conveniently but simply tagged.
      photos.copie
      To copy a set of files, slightly modifying the names. I use it to get a set of pictures that I want to edit, this explains its name.
      photos.reduis
      To reduce the size of a set of pictures, calling ImageMagick convert command.
      tabla.pl
      To perform transformations of csv data bases.
